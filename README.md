# AutoVPN

**Бесплатный VPN для Windows и Android**
Работает через протокол VLESS+Reality — один из самых быстрых и устойчивых к блокировкам.

---

## Скачать

> **[⬇ AutoVPN_v46.exe](https://github.com/artyomandreevich888-bit/AvtoVpn/releases/latest/download/AutoVPN_v46.exe)**

## Как пользоваться

1. Запусти приложение
2. Нажми кнопку питания — VPN подключится автоматически
3. Приложение само выберет самый быстрый сервер
4. Для отключения — снова нажми кнопку питания

## Часто задаваемые вопросы

**Почему Windows блокирует запуск?**
Это стандартное предупреждение для неподписанных приложений. Нажми "Подробнее" → "Всё равно запустить".

**Что такое Kill Switch?**
Если VPN внезапно отключится — Kill Switch полностью блокирует интернет, чтобы твой реальный IP не был виден. Включается в настройках (шестерёнка).

**Серверы не подключаются?**
Нажми кнопку обновления (🔄) в правом верхнем углу — приложение загрузит свежие серверы.
